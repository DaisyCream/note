###函数表达式的特征

- 定义函数的方式有两种：一种是函数声明，一种是函数表达式

- 函数声明提升：在执行代码之前会先读取声明的函数

函数声明：

```javascript
	sayHi();//Hi
	
	function sayHi(){
	    console.log("Hi");
	}
```
